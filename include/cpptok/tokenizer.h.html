19/*!
20 * \class Tokenizer
21 * \brief produces token from an input string
22 *
23 * The Tokenizer is a line-based tokenizer.
24 * That is, it is able to produce tokens for one line of input at a time and maintains
25 * a state to produce the correct output for multi-line constructs (currently that means
26 * multi-line comments).
27 *
28 * The tokenize() methods provide tokenization for a variety of inputs, but ultimately
29 * an array of char* is used.
30 *
31 * The output tokens are written in the \c output member of the class.
32 */
33
